# OpenglESUtils
 **此库是对opengles 的一些常用操作的封装。
   包括egl、opengl es线程、纹理加载操作、关联到texture的framebuffer等常用功能。
   
#NDK层编译步骤
   系统环境：Ubuntu<br>
   ndk版本：12b<br>
   cmake3.9以上<br>
   
   export NDKROOT=/ndkdir/ <br>
   mkdir thirdparts<br>
   cd thirdparts<br>
   git clone https://github.com/showwaychen/baselib.git<br>
   ./build.sh<br>
   cd ../openglesutils/openglcapture<br>
   ./build.sh<br>
